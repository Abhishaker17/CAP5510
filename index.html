Abstract

Motif Search

Motifs are approximate patterns found in gene sequences. Planted Motif Search can be described as a method of identifying conserved substring of length l that occurs in a set of input sequences with up to d mismatches. Many versions of motif search problems have been studied intensively, but we shall focus on PMS.

Protein Motif Search Problem

PMS, which has been well studied, is though to be a NP-hard problem. There are two types of algorithmic categories with regards to PMS i.e. approximation algorithms and exact algorithms. Approximation algorithms are generally faster than exact algorithms, which have an exponential worst case complexity. While exact algorithms for the motif problem take longer to complete, they guarantee to find all motifs. But due to their exponential time complexity, it is impractical to run these algorithms on very large instances.

Implementation

This project will implement a OS X based application implementing Protein Motif Search Prune algorithm, which is based on the idea of PMS1 algorithm. Current implementations take a console based approach. With the help of Cocoa API and the powerful Clang compiler, it will now be possible to group and visualize data in an intelligible manner.

Plan of Action

  • Understand the different motif sequence search algorithms through papers and available literature.
  • Correspond with the paper's authors if clarification is needed.
  • Implement the algorithm in Objective-C/Swift and compile it with the Clang compiler.
  • Test it against manual input of different sizes. After successful testing, test it with actual nucleotide sequences. Test it for correctness and result quality.
  • Compare the result and performance against existing implementations (obtainable through GitHub) such as: > - PMS1 > - qPMS2

Most of the effort will go into making this interactive, once the initial challenge of implementing the algorithm is completed.
