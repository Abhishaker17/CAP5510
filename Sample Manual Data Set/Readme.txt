1) Input file format:
Input file shoulb be in FASTA format. The file input_example.txt in this package is a sample input file.

2) How to run the program:
In the command line, type: 
PMSPrune.exe [input file]  [output file] [motif length] [d]

For example:
PMSPrune.exe input_example.txt  output_example.txt 13 4
